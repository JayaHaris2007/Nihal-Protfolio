# Niyal Ahmed N - Personal Portfolio

A professional, modern, and responsive personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Features
- **Responsive Design**: Looks great on mobile, tablet, and desktop.
- **Smooth Animations**: Powered by Framer Motion.
- **Modern UI**: Clean, minimalistic, and premium aesthetic.
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic components.

## Tech Stack
- React.js
- Tailwind CSS
- Framer Motion
- Lucide React (Icons)

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Customization
Edit the components in `src/components/` to update your information, images, and projects.
